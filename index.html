Create Account

Create an account on the Stellar network

Choose your way to play...

If Stellar were a universe, and it is, accounts would be the planets, stars, moons, and asteroids within that fertile space.

Less abstractly, accounts are the central data structure in Stellar — they hold balances, sign transactions, and issue assets. Accounts can only exist with a valid keypair and the required minimum balance of lumens (XLM).

In this quest, your challenge is to create an account by using the createAccount operation with the Quest Keypair located in the box on the right-hand side of this screen.

Let's get started!

  1. Start by clicking Register in the box on the right to receive your unique Quest Keypair, then click Fund to add testnet XLM to it.
  2. To begin, we're going to need a few things from the stellar-sdk.
const { Keypair, Server, TransactionBuilder, Networks, Operation, BASE_FEE } = require('stellar-sdk')
  1. Get your two keypairs ready — one for the Quest Account (from the sidebar), one for the new account you're creating.
const questKeypair = Keypair.fromSecret('SECRET_KEY_HERE') const newKeypair = Keypair.random()
  1. Set up a server and load the Quest Account from Horizon:
    • A server to retrieve and submit information to the network.
    • The account associated with your Quest Keypair.
// Remove '-testnet' when using the public network. const server = new Server('https://horizon-testnet.stellar.org') const questAccount = await server.loadAccount(questKeypair.publicKey())
  1. Build the transaction using TransactionBuilder.
let transaction = new TransactionBuilder(questAccount, { fee: BASE_FEE, networkPassphrase: Networks.TESTNET })
  1. Add the createAccount operation and sign.
transaction = transaction .addOperation(Operation.createAccount({ destination: newKeypair.publicKey(), startingBalance: "1000" })) .setTimeout(30) .build() transaction.sign(questKeypair)
  1. Submit to the network.
try { let res = await server.submitTransaction(transaction) console.log(`Transaction Successful! Hash: ${res.hash}`) } catch (error) { console.log(error, `. More details:\n`, JSON.stringify(error.response.data.extras, null, 2)) }

Assuming everything went well, you can click Verify on the right to see if you passed and collect your Stellar Quest NFT badge.
