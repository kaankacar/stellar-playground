StellarQuest Testnet
Level 1 — Quest 1

Create Account

Create a funded Stellar testnet account using the Stellar SDK. Every account on the Stellar network requires a minimum balance of XLM to exist.

Stellar Lab
Manual Code
1

Generate a keypair

Use Keypair.random() to generate a new public/secret key pair. The public key is your account address. Keep the secret key private — it controls your account.

2

Fund via Friendbot

On testnet, you can fund any new account for free using Friendbot — Stellar's testnet faucet. It sends 10,000 XLM to the address you provide, which also creates the account on-chain.

3

Verify the account

Load your account from Horizon using server.loadAccount(publicKey) and confirm it has a balance. If it loads without error, the account exists on the network.

4

Run the script

Click ▶ Run in the playground on the right. When it finishes, your public key will appear in the output.

✓

To complete this quest: copy your Public Key from the playground output and paste it into StellarQuest to verify.
